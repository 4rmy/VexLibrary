# VexLibrary
A custom odometry and autonomous control library for VEX Robotics, built on top of [PROS](https://pros.cs.purdue.edu/). Designed to give teams a reliable, easy-to-integrate foundation for competitive autonomous routines.

> ⚠️ **This library is currently a work in progress.** Features are being actively developed. See the [Roadmap](#roadmap) for current status.

## Getting Started
VexLibrary is built on PROS. Make sure you have the [PROS CLI](https://pros.cs.purdue.edu/v5/getting-started/) installed before proceeding.
